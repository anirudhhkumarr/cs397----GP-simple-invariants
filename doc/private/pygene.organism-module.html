Module pygene.organism

Implements classes for entire organisms

Organisms produce Gametes (think sperm/egg) via the .split() method.

Organisms can be mated by the '+' operator, which produces a child organism.

Subclasses of Organism must override the following methods:
  • fitness - returns a float value representing the organism's fitness - a value from 0.0 to infinity, where lower is better
Refer to module pygene.prog for organism classes for genetic programming.
